# 🦊 Kitsune-Den

Mods, tools, and server infrastructure for survival games — built by Ada, tested by chaos.

Most of what lives here is for **7 Days to Die**, with more survival game support on the way.

## 🔧 What's here

| Project | What it does | Install |
|---|---|---|
| [KitsunePaint](https://github.com/Kitsune-Den/KitsunePaint) | Web-based paint pack creator for 7D2D | web tool |
| [KitsunePaintUnlocked](https://github.com/Kitsune-Den/KitsunePaintUnlocked) | Breaks the 255 paint texture hard limit to 1023 | server + client |
| [KitsuneCommand](https://github.com/Kitsune-Den/KitsuneCommand) | Full server management suite | server only |
| [KitsuneKitchen7D](https://github.com/Kitsune-Den/KitsuneKitchen7D) | 16 new food & drink recipes, magazine-gated | server only |
| [KitsuneVehicleOverhaul](https://github.com/Kitsune-Den/KitsuneVehicleOverhaul) | Vehicle overhaul mod | server only |
| [Kitsune7Den](https://github.com/Kitsune-Den/Kitsune7Den) | 7D2D server dashboard | server only |
